Neovim Hotkeys

Compact printable cheat sheet by @Emruz_Hossain. Press the button or use the browser print dialog.

Navigation

Hotkey Mode Function
h,l,k,j n,x Left, Right, Up, Down
Alt + h,l,k,j n,x Move line/selection left,right,up,down
Space + qq n Quite all
Space + sj n Jump list

Hotkey Mode Function

Space + bD n Delete buffer and window
Space + bl n Delete buffers to left
Space + br n Delete buffers to right
Space + bp n Pin/unpin buffer
Space + bP n Delete unpin buffers
Ctrl + h,l,k,j n,x Go to (left,right,up,down) window
Ctrl + up,down,left,right n,x Adjust window size
Space + - n Split down
Space + | n Split right
Space + wd n Delete window
Space + wm n Toggle zoom
