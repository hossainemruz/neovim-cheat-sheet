Neovim Hotkeys

Compact printable cheat sheet by @Emruz_Hossain. Press the button or use the browser print dialog.

Navigation

Hotkey Mode Function
h,l,k,j n,x Left, Right, Up, Down
Alt + k,j n,i,v Move line/selection up,down
Space + qq n Quite all

Tab & Window Management

Hotkey Mode Function
Ctrl + h,l,k,j n,x Go to (left,right,up,down) window
Ctrl + up,down,left,right n,x Adjust window size
Shift + h,l n Move to prev,next buffer
Space + bb n Switch to other buffer
Space + bd n Delete buffer
Space + bo n Delete other buffers
Space + bD n Delete buffer and window
Space + - n Split down
Space + | n Split right
Space + wd n Delete window
Space + wm n Toggle zoom

Search & Replace

Hotkey Mode Function
/ n Start search
n,N n,x,o Next/Prev search result

Testing & Debugging

Hotkey Function
Super + , Dismiss latest notification

AI

Hotkey Function
Ctrl+Shift+Super+Space Pick a new theme

Editing

Hotkey Mode Function
gco n Add comment below
gcO n Add comment above
Space + cf n,x Format code
Space + uf n Toggle auto-format
Space + us n Toggle auto-spelling
Space + ul n Toggle line number
Space + uL n Toggle relative line number
Space + uD n Toggle dimming
Space + uz n Toggle zen mode

Git

Hotkey Mode Function
Space + gL n Git log (cwd)
Space + gb n Git blame (line)
Space + gf n Git current file history

Tools

Hotkey Mode Function
Ctrl + / n,t Terminal
Space + l n Lazy

File Management

Hotkey Mode Function
Space + fn n New file

Diagnostics

Hotkey Mode Function
Space + xl n Location list
Space + xq n Quickfix list
[q , ]q n Prev/next quickfix
[e , ]e n Prev/next error
[w , ]w n Prev/next warning
